# PRISM / WEBP CONVERTER

> **Industrial strength image processing tool for the modern web.** > _Privacy-focused. Client-side. Precision engineered._

## // OVERVIEW

PRISM is a high-performance, browser-based tool designed to convert bulk images (PNG, JPG, BMP) into the highly efficient **WebP** format. Built with a "Swiss Industrial" aesthetic, it prioritizes functionality, speed, and user privacy.

**Live Demo:** [Coming Soon]

## // CORE FEATURES

- **Client-Side Processing:** All conversions happen locally in your browser. No files are ever uploaded to a server.
- **Bulk Conversion:** Drag & drop multiple files for parallel processing.
- **Interactive Comparison:** Visual "Before/After" slider to verify quality loss against compression gains.
- **Dark Mode:** Precision-crafted dark theme for low-light environments (System Sync + Toggle).
- **Custom Quality Control:** Fine-tune compression levels (1-100%).
- **Zipper:** One-click "Download All" as a ZIP archive.

## // TECH STACK

- **Core:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Design:** Custom "Swiss Industrial" Design System (CSS Variables)
- **Libraries:**
  - `JSZip` (Batch downloading)
  - `Phosphor Icons` (UI iconography)
  - `Canvas Confetti` (Micro-interactions)

## // INSTALLATION & USAGE

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/dhanushrs1/prism-img-tool.git
    cd prism-img-tool
    ```

2.  **Run Locally:**
    Simply open the `index.html` file in any modern web browser. No build step or server required.

## // USAGE GUIDE

1.  **Select Files:** Drag images into the drop zone or click to browse.
2.  **Configure:** Adjust the Quality Slider (Default: 85%).
3.  **Convert:** Click `INITIALIZE_CONVERSION`.
4.  **Review:** Use the `Eye` icon to compare the Original vs. WebP result.
5.  **Download:** Save individually or download the entire batch as a ZIP.
